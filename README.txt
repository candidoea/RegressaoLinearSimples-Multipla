Neste repositório, há dois notebooks nos quais realizei regressão linear simples e múltipla com o dataset BostonHousing, presente na biblioteca Scikit-learn do Python.

No arquivo de regressão simples(BostonHousingDataset-taxaMediaOcupacao.ipynb), o modelo foi feito com base no método dos mínimos quadrados. Após isso, fiz algumas previsões com o modelo treinado

No arquivo de regressão múltipla (BostonHousingdatasetRegressaoMultipla.ipynb), analisei o valor-p para verificar quais features eram relevantes para serem usadas. 
Verifiquei isso com matrizes de correlação e evidenciei a multicolinearidade com autovalores e autovetores. 
Utilizei o método do Gradiente Descendente para encontrar os valores dos coeficientes de regressão que minimizam a função de custo e avaliei o modelo com o R².
